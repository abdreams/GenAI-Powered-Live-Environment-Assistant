🎯 WHAT YOU NEED TO DO IN AZURE PORTAL:

Step 1: CREATE AZURE OPENAI RESOURCE
─────────────────────────────────────
1. Go to https://portal.azure.com
2. Search for "Azure OpenAI"
3. Click "Create"
4. Fill in:
   - Resource Group: (create new or select existing)
   - Region: East US / West Europe (pick one close to you)
   - Name: my-genai-assistant (or your choice)
   - Pricing Tier: Standard S0
5. Click "Review + Create" → "Create"
6. Wait 2-3 minutes

Step 2: DEPLOY GPT-4 MODEL
───────────────────────────
1. Go to your new resource
2. Click "Go to Azure OpenAI Studio" button
   OR visit: https://oai.azure.com/
3. Click "Deployments" (left sidebar)
4. Click "Create new deployment"
5. Configure:
   - Select model: gpt-4 (or gpt-4-turbo if available)
   - Deployment name: gpt-4
   - Model version: Latest
6. Click "Create"
7. ⚠️ REMEMBER your deployment name!

Step 3: GET YOUR CREDENTIALS
─────────────────────────────
Go back to Azure Portal → Your Resource:

A. GET API KEY:
   1. Click "Keys and Endpoint" (left sidebar)
   2. Copy "KEY 1"
   
B. GET ENDPOINT:
   1. Same page
   2. Copy "Endpoint"
   3. Example: https://my-genai-assistant.openai.azure.com/
   4. ⚠️ Must end with /

C. API VERSION:
   Use: 2024-02-15-preview
